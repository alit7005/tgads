## Install

   To install everything in one command (useful for VPS deployment) on Ubuntu 16.04 x64 (Debian-based distros), use:
```bash
   git clone https://github.com/alit7005/tgAds.git && cd tgAds && bash TG 
```

## Create bot and Launch

   screen -S nohup ./TG 1
